# 2016级项目实训成果展示 

## 《西遇》 - H5方向

### 项目简介

一款以旅游分享和查看旅游攻略为主的小程序。

和志同道合的小伙伴去北京天安门，去天津的天津之眼，上海的外滩等等。这个世界很大，这个世界很小，一起玩，才能更好玩！得到最新的旅游资讯才能更好的完成旅行。
